# Bozkanat Autonomy

A modular autonomy stack for fixed-wing UAVs featuring computer vision, target tracking, mission management and autonomous payload delivery.

## Project Status

Under Development

## Current Features

- Raspberry Pi Camera Integration
- YOLO-based Object Detection
- Target Representation
- Mission Management
- Target Tracking (v1)

## Planned Features

- Pixhawk / MAVLink Communication
- Autonomous Payload Release
- Servo Controller
- Flight State Management
- Mission Logging

## Technologies

- Python
- Raspberry Pi
- Picamera2
- Ultralytics YOLO
- OpenCV
